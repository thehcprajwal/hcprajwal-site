# hcprajwal | Portfolio Website

A personal portfolio website simulating a terminal system initialization process. It emphasizes building systems, automation workflows, and AI-powered tools.

## Structure

This is a static HTML/CSS/JS website structured professionally:
- `index.html` - The main entry point containing the webpage structure and SEO optimizations.
- `assets/` - Contains all static assets categorized by type.
  - `css/style.css` - Custom styling separated from the HTML for better maintainability.
  - `js/main.js` - JavaScript code separated for clean architecture.
- Root directory contains various manifest and favicon files for robust multi-device display support.

## Running Locally

To run this website locally, you can use any standard HTTP server or run it directly in your browser.

Example using Python:
```bash
python3 -m http.server
```
Then visit `http://localhost:8000`

## Deployment

Since this project consists entirely of static assets, it can be hosted on GitHub Pages, Vercel, Netlify, Cloudflare Pages, or simply dropped into an S3 bucket or traditional web host.
